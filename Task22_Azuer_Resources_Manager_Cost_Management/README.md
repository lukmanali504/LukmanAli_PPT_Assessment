## Objective
Enable monitoring and diagnostics for Azure resources and view performance metrics.

## Steps

1. Open Azure Monitor
   - Go to Azure Portal → Monitor

2. Select a resource
   - Choose an existing virtual machine or storage account

3. Enable diagnostics
   - Open resource → Monitoring → Diagnostic settings
   - Click Add diagnostic setting
   - Enable logs and metrics
   - Send to Log Analytics (create workspace if required)
   - Save settings

4. View metrics
   - Go to Monitor → Metrics
   - Select the resource
   - Choose metric (example: CPU Percentage)

5. Analyze data
   - Observe graphs and performance data

## Result
Monitoring and diagnostics are enabled and performance metrics are visible for the selected Azure resource.
