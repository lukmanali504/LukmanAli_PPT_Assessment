## Objective

Understand service availability (SLA), resource lifecycle, and manage subscriptions in Azure.

## Steps

1. View SLA information

   * Open Azure Portal
   * Select any resource (example: Virtual Machine)
   * Go to Support + troubleshooting → SLA
   * Review uptime percentage and conditions

2. Manage subscription

   * Go to Subscriptions
   * Select active subscription
   * View usage and billing details

3. Resource lifecycle

   * Create a test resource (example: storage account)
   * Stop or delete the resource
   * Confirm resource removal from resource group

4. Verify

   * Check activity log for create and delete operations

## Result

SLA details are reviewed, subscription information is accessible, and resource lifecycle operations are successfully performed.
