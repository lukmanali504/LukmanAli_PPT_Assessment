## Objective
Create users and assign roles using Entra ID for identity and access management.

## Steps

1. Open Microsoft Entra ID
   - Go to Azure Portal → Microsoft Entra ID

2. Create user
   - Users → New user → Create new user
   - Enter username and name
   - Set password
   - Create user

3. Verify user
   - Confirm user appears in users list

4. Assign role
   - Go to Roles and administrators
   - Select role (example: Reader)
   - Click Add assignments
   - Select created user
   - Assign role

5. Test login
   - Use provided username and password to sign in

## Result
User is successfully created and assigned a role with appropriate access permissions.
