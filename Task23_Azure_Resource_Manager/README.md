## Objective
Create and manage resources using Resource Groups and view cost analysis.

## Steps

1. Create Resource Group
   - Open Azure Portal → Resource groups → Create
   - Enter resource group name
   - Select region
   - Create resource group

2. Deploy a resource
   - Create a virtual machine or storage account inside the resource group

3. View resources
   - Open the resource group
   - Verify deployed resources are listed

4. Open Cost Management
   - Go to Cost Management + Billing → Cost analysis

5. View cost usage
   - Select subscription
   - View usage charts and service-wise cost

## Result
Resources are organized within a resource group and cost usage is visible in the cost analysis dashboard.
